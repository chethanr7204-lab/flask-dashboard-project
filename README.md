# Flask Dashboard Project

## Description
A web application with user login authentication and dashboard using Flask.

## Features
- User Login & Registration
- Password hashing
- Session-based login
- Dashboard after login
- Hosted on Render

## Technologies Used
- Python (Flask)
- HTML, CSS
- SQLite
- GitHub
- Render

## Live Demo
https://flask-dashboard-project.onrender.com

## How It Works
1. User logs in
2. Data checked from database
3. Session created
4. Redirect to dashboard
